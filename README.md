# Netflix Clone

## Description
A fully functional Netflix clone built with Django to practice and showcase my skills in web development. This project includes user authentication, content streaming, and a dynamic interface similar to Netflix.

## Features
- User authentication (registration, login, logout)
- Content browsing and searching
- Video playback
- User profiles
- Admin panel for content management
- PostgreSQL for database management

## Installation
Instructions to get the project up and running on a local machine:
1. Clone the repository
    ```bash
    git clone https://github.com/andrelcunha/django_netflx.git
    ```
2. Navigate to the project directory
    ```bash
    cd django_netflx
    ```
3. Install pipenv if you haven't already
    ```bash
    pip install pipenv
    ```
4. Install the dependencies
    ```bash
    pipenv install
    ```
5. Activate the virtual environment
    ```bash
    pipenv shell
    ```
6. Apply migrations
    ```bash
    python manage.py migrate
    ```
7. Run the development server
    ```bash
    python manage.py runserver
    ```
